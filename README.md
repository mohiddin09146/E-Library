# E-Library: Digital Library Management System

A responsive, web-based Digital Library Management System tailored for engineering institutions. This platform categorizes and manages academic books across all engineering departments while offering distinct portals for students and administrators.

## 🚀 Features

### 👤 Student Portal
* **Multi-Department Access:** Browse and search through comprehensive book inventories for all engineering departments (Computer Science, Mechanical, ECE, Civil, Electrical, etc.).
* **Smart Catalog Search:** Real-time filtering and search operations to locate textbooks instantly.
* **Interactive Interface:** Easy navigation to view book details and availability.

### 🛡️ Admin Portal
* **Centralized Dashboard:** A responsive administrative panel to monitor and organize multi-department inventories.
* **Catalog Management:** Add, update, and manage book listings efficiently.
* **Access Control:** Dedicated, secure admin login to prevent unauthorized management.

### 💻 Technical Highlights
* **Authentication UI:** Distinct student and admin login interfaces with interactive frontend validation.
* **Fully Responsive:** Designed using fluid grids and flexible components to work flawlessly on desktops, tablets, and smartphones.

## 🛠️ Technologies Used

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Framework:** Bootstrap 5 (for responsive layouts and UI components)
* **Icons:** Font Awesome / Bootstrap Icons
